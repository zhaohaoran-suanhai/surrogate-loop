# 标量代理模型正式闭环实施计划

> **供 Agent 工作者使用：** 必须使用 `subagent-driven-development` 或 `executing-plans` 逐项执行本计划。步骤使用复选框（`- [ ]`）跟踪。

**目标：** 实现 `自然语言意图 → 受约束配置 → 数值数据 → 候选模型训练 → 验证集选模 → 测试集验收 → 产物保存 → 重载推理 → 域外拒绝` 的最小正式闭环。

**架构：** Codex 负责把自然语言意图写成白名单 JSON 配置；Python CLI 只执行确定性模块。科学计算、数据划分、模型注册表、评价、产物、推理和命令编排保持独立，并通过明确的数据类和函数签名连接。

**技术栈：** Python 3.11、Pydantic 2、NumPy、SciPy、scikit-learn、Matplotlib、joblib、pytest、Ruff、uv。

## 全局约束

- 只支持 `du/dt = gamma*u + 0.5*t`、`u(0)=0`、`t∈[0,1]` 和 `gamma -> u(1)`。
- 只接受白名单模板 `forced_reaction_scalar_endpoint_v1`；不得执行方程字符串。
- 正式工况数为 120/40/40，冒烟工况数为 24/8/8，随机种子为 `20260716`。
- 数值解使用 `DOP853`、`rtol=1e-10`、`atol=1e-12`，必须以 `1e-9` 绝对/相对容差通过解析解校验。
- 数据按不同 `gamma` 工况划分；归一化器只在训练集拟合；测试集只评估最终模型一次。
- 候选模型固定为 1/2/3 阶 PRS、GPR 和小型 MLP。
- 验收阈值固定为测试 NRMSE `<=0.03`、最大绝对误差 `<=0.01`。
- 仅加载本项目生成且清单哈希匹配的 joblib 文件；域外、NaN 和无穷输入在模型调用前拒绝。
- 用户可读文档、模型卡和 CLI 错误使用中文。

**接口：**

- `CaseDataset(gamma: NDArray[float64], target: NDArray[float64])`。

- `DatasetSplit(train_x, train_y, validation_x, validation_y, test_x, test_y)`。
- `split_dataset(dataset: CaseDataset, sampling: SamplingSpec) -> DatasetSplit`。

解析解在 `abs(gamma*time)<1e-4` 时使用：

```python
time**2 / 4 + gamma * time**3 / 12 + gamma**2 * time**4 / 48 + gamma**3 * time**5 / 240
```

其余情况使用：

```python
(np.expm1(gamma * time) - gamma * time) / (2 * gamma**2)
```

## 任务 3：模型注册表、验证集选模和测试验收

**文件：**

- 新建：`src/surrogate_loop/models.py`
- 新建：`src/surrogate_loop/evaluation.py`

**接口：**

- `build_candidates(seed: int, names: tuple[str, ...]) -> dict[str, RegressorMixin]`。
- `Metrics(rmse, nrmse, mae, max_absolute_error)`，提供 `to_dict()`。
- `SelectionResult(selected_name, selected_model, validation_metrics, test_metrics, accepted, failures)`。
- `compute_metrics(y_true, y_pred) -> Metrics`。
- `train_select_and_test(split, candidates, acceptance) -> SelectionResult`。

模型参数固定为：

- PRS：`StandardScaler -> PolynomialFeatures(degree, include_bias=False) -> Ridge(alpha=1e-8)`。
- GPR：`StandardScaler -> GaussianProcessRegressor(ConstantKernel*RBF, alpha=1e-10, normalize_y=True, n_restarts_optimizer=2)`。
- MLP：`StandardScaler -> MLPRegressor((32,32), activation="tanh", alpha=1e-4, max_iter=2000, early_stopping=True, validation_fraction=0.2)`。

排序键固定为 `(validation_nrmse, simplicity_index)`，简单度顺序为 `prs_1, prs_2, prs_3, gpr, mlp`。单个模型失败时记录中文错误并继续；全部失败时抛出 `RuntimeError`。

`--smoke` 要求配置中的 `mode` 为 `smoke`；不允许静默缩减正式配置。管线在任何失败时保留运行目录并写入 `status.json` 与 `error.json`。
