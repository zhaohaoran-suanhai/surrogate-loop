# 一维热传导神经算子闭环实施计划

> **面向执行 Agent：** REQUIRED SUB-SKILL: Use `executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking. 用户已选择在 `main` 分支内联执行，不建立额外分支或 worktree。

**目标：** 在保留现有标量闭环兼容性的前提下，实现一维瞬态热传导的数值数据生成、POD/GPR 基线、PyTorch DeepONet 训练、独立验收、可信产物和有界推理闭环。

**架构：** 新功能放入独立的 `surrogate_loop.operator` 子系统，科学问题实现集中在 `operator/heat1d`。现有标量命令保持不变，神经算子通过 `surrogate-loop operator` 命令组运行；解析解只校验数值标签，DeepONet 必须独立通过验收。

**技术栈：** Python 3.11、uv、NumPy、SciPy、scikit-learn、Pydantic 2、Matplotlib、joblib、PyTorch 2.9.0 CUDA 12.6 wheel、pytest、Ruff。

```python
def require_torch():
    try:
        import torch
    except ImportError as error:
        raise RuntimeError(
            "神经算子依赖未安装，请运行 uv sync --extra operator --all-groups"
        ) from error
    return torch


def resolve_device(requested: str):
    torch = require_torch()
    if requested == "cpu":
        return torch.device("cpu")
    if requested == "cuda":
        if not torch.cuda.is_available():
            raise RuntimeError("配置要求 CUDA，但 torch.cuda.is_available() 为 False")
        return torch.device("cuda")
    return torch.device("cuda" if torch.cuda.is_available() else "cpu")
```

from surrogate_loop.operator.heat1d.analytical import analytical_solution
from surrogate_loop.operator.heat1d.problem import HeatParameters, make_grid
from surrogate_loop.operator.heat1d.solver import solve_case


def test_analytical_solution_satisfies_initial_and_boundary_conditions() -> None:
    parameters = HeatParameters(0.1, 1.1, -0.2)
    x, t = make_grid(65, 51)
    field = analytical_solution(parameters, x, t)
    expected_initial = 1.1 * np.sin(np.pi * x) - 0.2 * np.sin(2 * np.pi * x)
    np.testing.assert_allclose(field[0], expected_initial, atol=1e-14)
    np.testing.assert_allclose(field[:, [0, -1]], 0.0, atol=1e-14)


def test_crank_nicolson_converges_toward_analytical_solution() -> None:
    parameters = HeatParameters(0.2, 0.8, 0.3)
    coarse_x, coarse_t = make_grid(65, 51)
    fine_x, fine_t = make_grid(129, 101)
    coarse_error = np.linalg.norm(
        solve_case(parameters, coarse_x, coarse_t)
        - analytical_solution(parameters, coarse_x, coarse_t)
    )
    fine_error = np.linalg.norm(
        solve_case(parameters, fine_x, fine_t)
        - analytical_solution(parameters, fine_x, fine_t)
    )
    assert fine_error / np.sqrt(fine_x.size * fine_t.size) < coarse_error / np.sqrt(

- [x] **Step 3：实现问题数据类型和解析解**

```python
@dataclass(frozen=True)
class HeatParameters:
    alpha: float
    amplitude_1: float
    amplitude_2: float


def make_grid(nx: int, nt: int) -> tuple[NDArray[np.float64], NDArray[np.float64]]:
    return np.linspace(0.0, 1.0, nx), np.linspace(0.0, 1.0, nt)


def analytical_solution(parameters, x, t):
    xx = x[None, :]
    tt = t[:, None]
    first = parameters.amplitude_1 * np.exp(-parameters.alpha * np.pi**2 * tt) * np.sin(
        np.pi * xx
    )
    second = parameters.amplitude_2 * np.exp(
        -4.0 * parameters.alpha * np.pi**2 * tt
    ) * np.sin(2.0 * np.pi * xx)
    return first + second
```

- [x] **Step 4：实现 Crank–Nicolson 推进**

使用 `scipy.linalg.solve_banded`。令 `r = alpha * dt / dx**2`，左矩阵内部对角为 `1+r`、上下对角为 `-r/2`；右矩阵对角为 `1-r`、上下对角为 `r/2`。每个时间步只求内部节点，边界始终写为零。拒绝非均匀网格、非有限参数、`alpha <= 0` 和不足三个空间节点。

```python
def _make_mlp(input_dim: int, width: int, layers: int, output_dim: int) -> nn.Sequential:
    modules: list[nn.Module] = []
    current = input_dim
    for _ in range(layers):
        modules.extend((nn.Linear(current, width), nn.Tanh()))
        current = width
    modules.append(nn.Linear(current, output_dim))
    network = nn.Sequential(*modules)
    for module in network.modules():
        if isinstance(module, nn.Linear):
            nn.init.xavier_uniform_(module.weight)
            nn.init.zeros_(module.bias)
    return network


def forward(self, branch, trunk):
    branch_features = self.branch_net(branch)
    trunk_features = self.trunk_net(trunk)
    return torch.einsum("bp,qp->bq", branch_features, trunk_features) + self.bias
```

构造函数校验输入维度、层数、宽度和 latent dim 为正数，并校验 forward 的二维形状和最后一维。
